# Pizza-Sale-Analysis-2015
End To End Project (SQL  + Python + Power BI)
Power Bi Dashboard Link(https://drive.google.com/file/d/1uFGfTLEu_h792tYhmNF4wYDsxJMBe1Hv/view?usp=drive_link)
​### Project Overview
​This project is a comprehensive analysis of a pizza store's sales data for the year 2015.
The goal was to transform raw data into actionable business insights using a modern data stack. 
I handled everything from database management and SQL querying to interactive dashboard design.
​### Tech Stack
python(Numpy,Pandas,Matplotlib,sqlalchemy ( create_engine))
​Database: PostgreSQL (Data Storage & Advanced SQL Analysis)
​BI Tool: Power BI Desktop (Data Modeling & Visualization)
​Query Language: SQL (Postgres dialect)
​Reporting: DAX (Data Analysis Expressions) for custom KPIs
​​### Key Business Insights (The "Story")
​1. Sales Performance & Profitability
​Total Revenue: Generated $817.86K from over 21,000 orders.
​Total Pizzas Sold: A massive 50,000 pizzas were delivered throughout the year.
​Average Order Value (AOV): On average, each customer spent $38.31 per order.
​2. Consumer Preferences (Category Leaders)
​Classic Category is the crowd favorite, contributing 26.9% ($220K) of total revenue.
​Individual Champion: The Thai Chicken Pizza stands as the highest revenue-generating pizza, bringing in $43,434.
​Operational Trends (Peak Times)
​Seasonal Peak: July recorded the highest monthly revenue ($72.5K), followed by May and March.
​Hourly Rush: SQL analysis confirmed peak order times are 12:00 PM - 1:00 PM (Lunch) and 6:00 PM (Dinner).
​The October Dip: Identified a significant sales drop in October, presenting an opportunity for targeted marketing.
​### Strategic Recommendations
​Dynamic Staffing: Increase kitchen and delivery staff during the 12-1 PM and 6-8 PM windows to maintain service quality during peaks.
​Targeted Promotions: Launch a "Fall Festival" campaign in October to bridge the identified revenue gap.
​Inventory Management: Scale up ingredient stock for the Classic and Chicken categories, as they drive the bulk of the revenue.
